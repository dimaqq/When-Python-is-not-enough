When Python is not enough
=========================

[Slides for this talk](http://http://dimaqq.github.io/When-Python-is-not-enough)

Covered in this talk:

 * ipython
 * numpy
 * cython
 * ctypes
 * cffi
 * psyco
 * pypy

Install these packages:

 * ipython
 * pyreadline (win)
 * numpy
 * matplotlib (opt)
 * cython
 * cffi
 * psyco (py2.6)
 * pypy (separately)
